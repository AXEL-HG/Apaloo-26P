# Apaloo-26T 🚀

Este repositorio contiene las prácticas, ejercicios y proyectos desarrollados durante el trimestre **26-T** para la materia de **Apaloo** en la **UAM**. El objetivo principal es consolidar los conceptos de programación orientada a objetos, estructuras de datos y buenas prácticas de desarrollo de software utilizando **Java**.

---

## 📂 Estructura del Repositorio

El proyecto está organizado de la siguiente manera para mantener el código limpio y fácil de navegar:

* **`practicas/`**: Carpetas individuales para cada una de las entregas semanales de la materia.
* **`proyectos/`**: Proyectos principales o finales desarrollados a lo largo del curso (tanto individuales como colaborativos).

---

## 🛠️ Tecnologías Utilizadas

* **Lenguaje:** Java
* **Control de Versiones:** Git & GitHub
